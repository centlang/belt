# Belt Browser

Proof of concept WebKitGTK-based browser.

![Preview](assets/preview.png)

## Building

Make sure you have the dependencies installed:

- Cent
- WebKitGTK
- Make

Clone the repository:

```sh
$ git clone https://github.com/centlang/belt && cd belt
```

Build:

```sh
$ make
```

## Running

```sh
$ build/release/belt
```
